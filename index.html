Caso de éxito · Modernización de plataforma de datos

From Data Chaos to an AI-Ready Lakehouse

Modernización de una plataforma de datos en retail / supply chain alrededor de Infor M3. El proyecto transformó un ecosistema fragmentado, con alta dependencia de Azure SQL y procesos aislados, en una base Lakehouse gobernada con Databricks, Delta Lake, Unity Catalog, capas Bronze/Silver, reconciliación automática, backfill y data products por dominio.

80+tablas procesadas con un framework dinámico basado en configuración.
75%–98%reducción del tiempo de procesamiento por tabla: de 2–4 horas a 5–30 minutos, según volumen de archivos.
80%+reducción de costo al reemplazar Dataflows por Copy Activities sobre Delta.
100%reconciliación Bronze vs Infor Compass para entidades validadas.

Reto

  • Procesos tabla por tabla difíciles de mantener.
  • Dependencia fuerte de Azure SQL y vistas complejas.
  • Falta de snapshots históricos, trazabilidad y recuperación.

Solución

  • Arquitectura medallion con Bronze append-only y Silver deduplicado.
  • Framework reusable de ingesta, deduplicación y backfill.
  • Lakeflow pipelines y data products por dominio para operación y analítica.

Impacto

  • Menor costo e infraestructura más eficiente.
  • Mayor confianza con reconciliación y controles de calidad.
  • Base gobernada y preparada para analítica avanzada e IA.

Success story · Data platform modernization

From Data Chaos to an AI-Ready Lakehouse

Retail / supply chain data platform modernization around Infor M3. The project transformed a fragmented, Azure SQL-heavy estate with isolated processes into a governed Lakehouse foundation using Databricks, Delta Lake, Unity Catalog, Bronze/Silver layers, automated reconciliation, backfill capabilities and domain-oriented data products.

80+tables processed through a dynamic configuration-driven framework.
75%–98%reduction in processing time per table: from 2–4 hours to 5–30 minutes, depending on file volume.
80%+cost reduction by replacing Dataflows with Copy Activities over Delta.
100%Bronze vs Infor Compass reconciliation for validated entities.

Challenge

  • Table-by-table processes that were hard to maintain.
  • Heavy Azure SQL dependency and complex views.
  • No reliable historical snapshots, traceability or recovery path.

Solution

  • Medallion architecture with append-only Bronze and deduplicated Silver.
  • Reusable ingestion, deduplication and backfill framework.
  • Lakeflow pipelines and domain data products for operations and analytics.

Impact

  • Lower cost and more efficient infrastructure.
  • Higher trust through reconciliation and data quality controls.
  • Governed foundation ready for advanced analytics and AI.
